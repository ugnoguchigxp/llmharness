# 改善依頼書パック（2026-04-10）

本ディレクトリは、`llmharness` を「単発パイプライン」から「フィードバックループ付きシステム」へ進化させるための、各プロジェクト向け依頼書です。

対象プロジェクト:

1. `llmharness`（Orchestrator）
2. `Astmend`
3. `DiffGuard`
4. `localLlm`
5. `gnosis`

共通ゴール:

1. DiffGuard の指摘を次回生成へ返す再試行ループを成立させる
2. 試行状態（attempt / issue / reject）を構造化し、追跡可能にする
3. 失敗と成功を gnosis に蓄積し、次回生成品質を改善する

各依頼書は以下を含みます:

1. 背景
2. 依頼内容（MVP）
3. 期待I/F
4. 受入基準（DoD）
5. 成果物
